# Boolean-Calculator

A simple boolean calculator written for the Assembly Language Programming class at Anoka Ramsey Community college.

The calculator supports the following operations on 32-bit operands:
  - Logical And
  - Logical Or
  - Logical Not
  - Logical Xor
